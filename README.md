# zapadniy-front

Frontend-часть проекта `ZOV`. Репозиторий содержит клиентское приложение, которое отвечает за пользовательский интерфейс системы, работу с картой, визуализацию данных и взаимодействие с backend-сервисами.

## Что делает проект

- предоставляет интерфейс для авторизации и работы пользователя в системе;
- отображает профиль пользователя и связанные с ним данные;
- показывает ближайших пользователей и взаимодействия между ними;
- визуализирует регионы и события на карте;
- поддерживает работу с supply chain и отображение маршрутов;
- получает данные от backend по HTTP и через `WebSocket`.

## Технологии

- `React`
- `TypeScript`
- `Vite`
- `Tailwind CSS`
- `Axios`
- `Leaflet`
- `React Leaflet`
- `STOMP`
- `SockJS`

## Структура

Основное приложение находится в каталоге `zapadniy-front/`.

## Запуск

```bash
cd zapadniy-front
npm install
npm run dev -- --host
```
