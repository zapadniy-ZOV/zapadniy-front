# zapadniy-front

Клиентское приложение для системы `ZOV`. Этот проект отвечает за пользовательский интерфейс, отображение карты, работу с регионами, профилями пользователей и визуализацию данных, которые приходят от backend-сервисов.

## Что делает проект

- предоставляет интерфейс входа в систему;
- отображает профиль пользователя и его текущие данные;
- показывает список ближайших пользователей;
- позволяет работать с социальными взаимодействиями внутри системы;
- отображает карту регионов и выбранные области;
- визуализирует цепочки поставок и связанные маршруты;
- получает обновления от backend в реальном времени.

## Запуск

1. Установить зависимости:

```bash
npm install
```

2. Запустить dev-сервер:

```bash
npm run dev -- --host
```

3. Собрать production-версию:

```bash
npm run build
```
